# Sunil Kumar Portfolio (Next.js)

Premium futuristic portfolio built with Next.js + Tailwind + Framer Motion + TypeScript.

## Features
- Multi-page architecture (Home, Projects, Resume, Contact)
- Glassmorphism neon blue/purple dark theme
- Responsive navbar and sections
- Animated hero, floating profile, motion transitions
- Skills, projects, experience, achievements, coding profiles
- SEO metadata, favicon, reusable component structure

## Run
```bash
npm install
npm run dev
```

## Deploy
- Push to GitHub
- Import to Vercel and deploy
